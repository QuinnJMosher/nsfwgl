# nsfwgl
Not suited for work, openGL.

-Controlls-
	-camera-
		W, S, A, D - pan up, down, left, and right (respectively)
		E + mouse - tilt camera
	-debug-
		Z - toggle specular lighting
		X - toggle diffuse lighting
		C - toggle Ambient lighting
		V - toggle Shadows
		B - toggle Color
		N - toggle "SoulSpear" normal map
		M - toggle particles
